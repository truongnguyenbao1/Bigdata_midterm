This project, part of the Introduction to Big Data course at 
Ton Duc Thang University, involves implementing three task 
to explore Big Data tech-niques. Conducted by a group of four 
students over two weeks, the project uses Python in Visual 
Studio Code and Google Colab, with the report compused in 
Overleaf using the IEEE template. The tasks include: (1) Set up 
a Hadoop system in the fully-distributed mode, use Docker 
containers for execute the Word Count job to verify your 
installation and implement the A-Priori algorithm to identify 
frequent pairs in form of Hadoop MapReduce programs; (2) 
Use RDD (without using DataFrame and with using 
DataFrame) and apply PCY algorithms to complete the 
problem requirements; (3) Using the FPGrowth algorithm, 
existing in PySpark , to discover frequent pairs and association 
rules based on a given support threshold s and a confidence 
threshold c.# Bigdata_midterm
